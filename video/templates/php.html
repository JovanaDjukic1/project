PHP

PHP je jedan od najpopularnijih programskih jezika današnjice za izradu web aplikacija i mnogih drugih softverskih produkata. Tvorac PHP-a je Rasmus Lerdorf a nastao je 1994.godine. Razvio se iz programskog jezika Perl. Kroz dalji period PHP je doživio značajne promjene a danas je on stabina, prilagodjena, funkcionalna cjelina sa ogromnom programerskom podrškom. Poslednja dostupna verzija je PHP 8 i besplatna je.

PHP je open-source skriptni programski jezik. Njegova glavna primjena je izrada dinamičkog web sadržaja. PHP kod se može direktno upisati u HTML sadržaj. Interpreter prolazi kroz sadržaj web stranice, pronalazi PHP naredbe koje izvršava a nakon toga se web sadržaj vraća klijentu. Spada u grupu imperativnog i objektno orjentisanog tipa. Koristi se često u kombinaciji sa Apache web serverom. Podržava ISAPI (Internet Server Application Program Interface) protokol koji se koristi za razvoj aplikacija. Takodje podržava i rad sa bazama podataka: MySQL, PostgreSQL, Oracle, SQLite, InfomixSybase i dr.PHP datoteke prepoznajemo po ekstenzijama .php, .php3 i .phtml i one mogu sadžati tekst, skripte i HTML tagove.

Koristi se kao:

Skriptovanje od strane servera,

Skriptovanje komandne linije,

Pravljenje GUI aplikacija od strane klijenata.

Sintaksa PHP-a je slična sintaksi C-a i Perl-a. Kod PHP-a je odvojen u posebne tagove. Naredbe u tagovima se medjusobno uvijek odvajaju sa ";". Jezik PHP spada u grupu case-sensitive jezika (jezici osjetljivi na veličinu slova). Imena varijabli počinju sa znakom dolar. Prvi karakter nakon dolara može biti slovo ili podvlaka a preostali dio može biti sačinjen od slova, brojeva ili podvlake. Vrijednosti konstanti se definišu pomoću funkcije define koja prima dva argumenta, naziv konstante i njenu vrijednost.

Postoje tri tipa podataka. To su:

1.skalarni:integer, float, string, Boolean -tipovi kao u većini programskih jezika

2.kolekcije:array, object

3.posebni tipovi:izvori,NULL.

Postoje ugradjene funkcije kao što su echo, if, while, ... a korisnik može proizvoljno definisati svoje funkcije.
